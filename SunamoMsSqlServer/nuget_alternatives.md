# NuGet Alternatives to SunamoMsSqlServer

This document lists popular NuGet packages that provide similar functionality to SunamoMsSqlServer.

## Overview

SQL Server utilities

## Alternative Packages

### System.Data.SqlClient
- **NuGet**: System.Data.SqlClient
- **Purpose**: ADO.NET SQL Server provider
- **Key Features**: Direct SQL Server access

### Microsoft.Data.SqlClient
- **NuGet**: Microsoft.Data.SqlClient
- **Purpose**: Modern SQL Server provider
- **Key Features**: Latest features, actively maintained

### Dapper
- **NuGet**: Dapper
- **Purpose**: Micro-ORM
- **Key Features**: Fast data mapping, simple queries

### Entity Framework Core
- **NuGet**: Microsoft.EntityFrameworkCore.SqlServer
- **Purpose**: Full ORM
- **Key Features**: Code-first, migrations, LINQ

## Comparison Notes

Microsoft.Data.SqlClient for ADO.NET. Dapper for micro-ORM. EF Core for full ORM.

## Choosing an Alternative

Consider these alternatives based on your specific needs:
- **System.Data.SqlClient**: ADO.NET SQL Server provider
- **Microsoft.Data.SqlClient**: Modern SQL Server provider
- **Dapper**: Micro-ORM
